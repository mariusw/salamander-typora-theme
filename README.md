# Salamander Typora theme

A warm dark Typora theme with oranges and browns in various hues. 

**Palette:** Dark brown background (`#0E0A06`), warm beige text, burnt orange accents.  
**Fonts:** Syne (headings) · Plus Jakarta Sans (body) · JetBrains Mono (code) — all bundled, works offline.

## Install

1. Download the [latest release](https://github.com/mariusw/typora-salamander/releases/latest) zip
2. Put `salamander.css` and the `salamander` folder in your Typora themes folder (`Preferences → Appearance → Open Themes Folder`)
3. Select **Salamander** in Preferences → Appearance → Theme
